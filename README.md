# Network game Tic-Tac-Toe based on WPF and WCF 

To invite to the game click right mouse button on the user who is online.

- WPF;
- WPF vector graphics;
- WCF;
- MS SQL DB;
- LINQ to SQL;
- MVVM;
- Custom styles;
- Custom window;
- Game lobby with chat;
- Game history and mini stats of win/lose;
- Client/Server model.
